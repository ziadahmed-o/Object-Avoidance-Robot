# 🤖 Autonomous Object-Avoidance Robot

A fully functional autonomous robot designed to navigate environments while detecting and avoiding obstacles in real-time. This project integrates **Arduino microcontrollers**, ultrasonic sensor arrays, and motor driver logic to achieve intelligent obstacle navigation.

## 📖 Technical Implementation

* **Obstacle Detection:** Utilizes ultrasonic sensors to map surrounding distances and trigger avoidance maneuvers.
* **Logic & Control:** Programmed using **Arduino C++** with specific algorithms for motor speed control and directional logic.
* **Hardware Setup:** 
    * Modular chassis design.
    * H-Bridge motor control for precise movement.
    * Integrated wiring and power management.
